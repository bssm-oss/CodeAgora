# examples/

## Responsibility
Demonstration applications used to exercise CodeAgora against realistic, intentionally insecure code.

## Design
Examples are isolated from the main product surface and may use different dependencies or patterns to showcase detection coverage and reviewer behavior.

## Flow
Demo code is fed into the review pipeline as sample input, producing findings that validate security and quality detectors.

## Integration
Supports tests, demos, and detector development; the example outputs are not part of production runtime paths.
