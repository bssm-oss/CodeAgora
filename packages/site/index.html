주장마다 근거를 붙이고 약한 추측은 낮춥니다.

없는 파일, 틀린 라인, 잘못된 인용을 걸러냅니다.

02

서로 다른 관점을 한곳에 모읍니다

03

결정 피로를 낮춥니다

읽을거리보다 지금 할 일을 앞에 둡니다.

CodeAgora는 무엇인가요?

여러 AI 리뷰어가 같은 diff를 보고 근거 있는 판정으로 정리하는 코드 리뷰 도구입니다.

하나의 AI 코드 리뷰와 무엇이 다른가요?

한 모델의 답을 그대로 믿지 않고, 여러 주장과 근거를 비교합니다.

GitHub Action 코드 리뷰 자동화에 쓸 수 있나요?

네. Pull Request마다 같은 리뷰 기준을 자동으로 붙일 수 있습니다.

MCP는 왜 필요한가요?

AI IDE와 에이전트가 CodeAgora를 정해진 도구 계약으로 호출하게 합니다.

API 키와 코드는 어디에 저장되나요?

API 키는 로컬 권한 안에서 관리하고, 비밀값은 로그와 산출물에서 가립니다.

Desktop은 안정판인가요?

아직 RC 경로입니다. 안정판 Desktop 배포 약속은 아닙니다.

주장은 테스트로 확인합니다.
